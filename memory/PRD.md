# High Square Aluminium - Showcase Website

## Original Problem Statement
Premium, ultra-modern showcase website for "High Square Aluminium by Monalisa Aluminium" — a luxury aluminium brand serving Indore & Dewas since 1998. Deep matte black + metallic gold theme inspired by hline.in.

## Architecture
- **Frontend**: React + Tailwind CSS + Framer Motion (static, no backend)
- **Form Service**: Formspree (direct email delivery)
- **Hosting**: GitHub Pages / Vercel / Netlify (static)

## User Personas
- Architects seeking premium aluminium partners
- Homeowners looking for window/door solutions
- Builders/contractors evaluating aluminium suppliers

## Core Requirements
- 10-section single-page showcase website
- Contact form with MongoDB storage
- Premium animations and visual effects
- Mobile responsive design

## What's Been Implemented (Dec 2025)
- Hero section with fullscreen architecture background
- Brand statement with large bold typography
- About section with company history and stats
- Services grid (6 services: Sliding/Casement/Domal Windows, Doors, Partitions, Invisible Grills)
- Product showcase with 3 categories (Windows, Doors, Partitions)
- Why Choose Us with 5 differentiators
- Projects masonry gallery with lightbox
- Custom Solutions section
- Contact form (saves to MongoDB via /api/contact)
- Footer with quick links
- Navbar with mobile menu
- Framer Motion animations throughout
- CSS 3D tilt effects on cards
